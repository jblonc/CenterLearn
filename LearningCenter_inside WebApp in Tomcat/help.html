To generate a new user id: Type the ID into the the box 'Enter LnCtr ID', click 'record' button; the student can now logon using the ID and the password 'learningcenter'; when logged on, the password can and should be changed by the student (It's recommended that email address without domain address serves as Learning Center ID, but it can be anything else that distinguishes students)

To enter the time a student comes in: Type the student's ID into the box 'Enter LnCtr ID'; (must) select a course name from the dropdown; (must) select a professor; click 'record' button. This automatically registers the current time as the student's coming-in time; if the current time is not the student's coming-in time, simply adjust the time next to 'Start Time'. The date in 'Start Time' must be today's date (so don't do anything about the date). If the student doesn't already have an ID, this also automatically creates an ID for the student.

To enter the complete record of a student's visit: Type the student's ID; (must) select a course name; (must) select the professor; choose 'Start Time'; switch 'End Time? No' to 'End Time? Yes'; choose end time; choose a tutor (optional); click 'record' button. The dates for start time and end time must be the same, however, they don't have to be today's date. The end time must be later than the start time. If the student doesn't already have an ID, this also automatically creates an ID for the student.
